Projektaufgabe

Durch die Webservice-Tutorials ist nun bekannt, wie Webservices sowohl mit HTTP GET als auch HTTP PUT genutzt werden. Nun müssen die vom Questionnaire-Server bereitgestellten Services vom Client konsumiert werden. Am Ende sollte die Anwendung eine Übersicht der vorhandenen Umfragen anzeigen. Aus diesen kann der Nutzer eine Auswählen, um daran teilzunehmen. Die Formulare für die Umfrage muss dynamisch an Hand der vom Webservice gelieferten Daten generiert werden. Vor dem endgültigen Absenden der vom Nutzer eingegebenen Daten, müssen diese mit dem entsprechenden Service validiert werden.

Der Service zur Ermittlung der Umfragestatistiken muss noch nicht integriert werden.
