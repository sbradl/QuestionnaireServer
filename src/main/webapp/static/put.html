Daten senden

Nachdem eingegebene Daten verifiziert wurde, können sie zur Auswertung an den Server gesendet werden. Dies geschieht ebenfalls mit einem HTTP PUT Request. Wie auch bei der Datenverifizierung muss der Content-Type angegeben werden. Als URL für das Hochladen der Daten wird http://localhost:8080/questionnaire/put.xml verwendet.
